사용된 HTML 속성 설명
  • colspan="숫자": 가로로 병합할 셀의 개수
  • rowspan="숫자": 세로로 병합할 셀의 개수
  • border-collapse: collapse: 테이블 테두리를 하나로 합침
  • th: 테이블 헤더 셀
  • td: 테이블 데이터 셀
